# Módulo de Asistencia (Attendance)

## 1. Propósito

El módulo de **Asistencia** proporciona a los entrenadores y al personal del club una herramienta para registrar y gestionar la asistencia de los socios a clases, entrenamientos y eventos programados.

## 2. Funcionalidades Principales

-   **Toma de Asistencia:** Los entrenadores pueden acceder a una lista de los socios inscritos en una clase o grupo de entrenamiento y marcar fácilmente quiénes están presentes, ausentes o llegaron tarde.
-   **Historial de Asistencia:** Tanto los socios como el personal administrativo pueden consultar el historial de asistencia de un miembro. Esto es útil para:
    -   Seguimiento del progreso y compromiso del socio.
    -   Verificar el cumplimiento de requisitos de asistencia para ciertos programas.
    -   Generar informes sobre la popularidad de las clases.
-   **Integración con Reservas:** El sistema muestra automáticamente la lista de asistentes que han reservado su lugar a través del módulo de **Reservas (Booking)**.
-   **Informes y Estadísticas:** Genera informes sobre la tasa de asistencia por clase, disciplina o entrenador, ayudando al club a optimizar la oferta de horarios y recursos.
