# Módulo de Equipos (Team)

## 1. Propósito

El módulo de **Equipos** está diseñado para que los socios puedan crear, gestionar y unirse a equipos para participar en los torneos organizados por el club. Está estrechamente ligado al **Módulo de Campeonatos (Championship)**.

## 2. Funcionalidades Principales

-   **Creación de Equipos:** Un socio puede crear un nuevo equipo, asignarle un nombre, un logotipo y convertirse en su capitán.
-   **Gestión de Miembros del Equipo:**
    -   **Invitaciones:** El capitán del equipo puede enviar invitaciones a otros socios del club para que se unan a su equipo.
    -   **Aceptación/Rechazo:** Los socios reciben las invitaciones y pueden aceptarlas o rechazarlas.
    -   **Lista de Jugadores (Roster):** El capitán puede gestionar la lista de jugadores de su equipo.
-   **Inscripción a Campeonatos:** Una vez que un equipo tiene el número mínimo de jugadores requerido, el capitán puede inscribirlo en un campeonato abierto.
-   **Perfil del Equipo:** Cada equipo tiene una página de perfil donde se puede ver su lista de jugadores, los campeonatos en los que participa y su historial de partidos.
-   **Comunicación de Equipo (Próximamente):** Funcionalidades para que los miembros de un equipo puedan comunicarse entre sí.
