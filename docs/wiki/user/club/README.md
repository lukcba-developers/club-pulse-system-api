# Módulo de Club (Club)

## 1. Propósito

El módulo de **Club** centraliza la gestión de la información institucional y la configuración general del club. Actúa como el panel de control principal para los administradores.

## 2. Funcionalidades Principales

-   **Gestión de Información del Club:** Permite a los administradores editar datos clave como:
    -   Nombre del club.
    -   Dirección y datos de contacto.
    -   Horarios de apertura.
    -   Logotipo y branding.
-   **Configuración de Módulos:** Desde este módulo, los administradores pueden habilitar, deshabilitar o configurar parámetros de otros módulos. Por ejemplo, establecer las políticas de cancelación de reservas del **Módulo de Reservas**.
-   **Gestión de Noticias y Anuncios:** Permite publicar noticias o anuncios importantes que serán visibles para todos los socios en el panel principal del frontend.
-   **Administración de Sedes:** Si el club tiene múltiples sedes o sucursales, este módulo permite gestionarlas de forma centralizada. Cada sede puede tener sus propias instalaciones y horarios.
