# Wiki de Usuario - Club Pulse System

Bienvenido a la documentación para usuarios del sistema Club Pulse.

Esta wiki proporciona una guía detallada sobre todas las características, casos de uso y flujos de trabajo dentro del sistema. La documentación está organizada por módulos funcionales para facilitar la consulta.

## Módulos

A continuación, se presenta la lista de módulos documentados. Haz clic en cualquiera de ellos para acceder a su documentación específica.

*   [**Acceso (Access)**](./access/README.md) - Control de entrada y salida de las instalaciones.
*   [**Asistencia (Attendance)**](./attendance/README.md) - Registro de asistencia a clases y eventos.
*   [**Autenticación (Auth)**](./auth/README.md) - Gestión de inicio de sesión, roles y permisos.
*   [**Campeonatos (Championship)**](./championship/README.md) - Gestión de torneos, partidos y clasificaciones.
*   [**Reservas (Booking)**](./booking/README.md) - Sistema de reservas de instalaciones y clases.
*   [**Club (Club)**](./club/README.md) - Gestión de la información general del club.
*   [**Disciplinas (Disciplines)**](./disciplines/README.md) - Gestión de deportes y grupos de entrenamiento.
*   [**Equipos (Team)**](./team/README.md) - Gestión de equipos para campeonatos.
*   [**Instalaciones (Facilities)**](./facilities/README.md) - Gestión de las instalaciones deportivas.
*   [**Incidentes (Incidents)**](./incidents.md) - Registro de incidentes y accidentes.
*   [**Gamificación (Gamification)**](./gamification.md) - Sistema de insignias, XP y tablas de clasificación.
*   [**GDPR (GDPR)**](./gdpr.md) - Derechos de portabilidad y eliminación de datos.
*   [**Grupos Familiares (Family)**](./family/README.md) - Vinculación de socios y facturación unificada.
*   [**Membresías (Membership)**](./membership/README.md) - Administración de tipos de membresía, socios y planes.
*   [**Notificaciones (Notification)**](./notification/README.md) - Sistema de envío de notificaciones a los usuarios.
*   [**Pagos (Payment)**](./payment/README.md) - Procesamiento de pagos y gestión de facturas.
*   [**Tienda (Store)**](./store/README.md) - Venta de productos, merchandising y artículos del buffet.
*   [**Usuarios (User)**](./user/README.md) - Gestión de perfiles de usuario.
