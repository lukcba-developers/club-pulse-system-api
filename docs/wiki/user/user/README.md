# Módulo de Usuario (User)

## 1. Propósito

El módulo de **Usuario** gestiona toda la información relacionada con los perfiles de los socios. Es el punto central para los datos personales y la configuración de la cuenta de cada miembro del club.

## 2. Funcionalidades Principales

-   **Gestión de Perfil:** Los socios pueden ver y editar su información personal, como:
    -   Nombre y apellidos.
    -   Foto de perfil.
    -   Datos de contacto (teléfono, dirección).
    -   Información de emergencia.
-   **Gestión de Grupos Familiares:**
    -   Permite a un socio (cabeza de familia) agrupar varias cuentas de miembros bajo un mismo grupo familiar.
    -   Esto es útil para gestionar las membresías y pagos de los hijos u otros familiares de forma centralizada.
    -   El cabeza de familia puede tener permisos para realizar reservas o pagos en nombre de los otros miembros del grupo.
-   **Preferencias de Comunicación:** Los usuarios pueden elegir cómo desean recibir las notificaciones del club (ej: por email, SMS), gestionado a través del **Módulo de Notificaciones**.
-   **Visualización de Actividad:** El perfil de usuario sirve como un panel central donde el socio puede ver un resumen de su actividad reciente: próximas reservas, últimas facturas, etc.
