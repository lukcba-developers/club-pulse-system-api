# Módulo de Acceso (Access)

## 1. Propósito

El módulo de **Acceso** es responsable de gestionar y validar el ingreso físico de los socios a las instalaciones del club. Su objetivo principal es asegurar que solo los miembros con membresías activas y permisos adecuados puedan acceder a las áreas permitidas.

## 2. Funcionalidades Principales

-   **Validación de Acceso:** El sistema se integra con hardware de control de acceso (como lectores de códigos QR, tarjetas NFC o torniquetes) para validar en tiempo real el estado de un socio.
-   **Registro de Entradas y Salidas:** Cada intento de acceso, ya sea exitoso o denegado, queda registrado en el sistema. Esto permite llevar un historial completo de quién ha estado en el club y cuándo.
-   **Reglas de Acceso:** Permite configurar reglas de acceso personalizadas. Por ejemplo:
    -   Restringir el acceso a ciertas instalaciones (ej: gimnasio, piscina) según el tipo de membresía.
    -   Limitar el acceso por horarios específicos.
    -   Validar si el socio tiene una reserva activa para una instalación en particular.
-   **Notificaciones de Acceso:** En caso de un intento de acceso denegado, el sistema puede notificar al personal del club para que puedan asistir al socio.
